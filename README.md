# 🤖 ChatBot Android App

An AI-powered ChatBot Android application built using **Kotlin** and **Jetpack Compose** that enables real-time conversational interaction.
The app integrates **Firebase AI API** to generate intelligent responses dynamically, providing a realistic chatbot experience.

---

## ✨ Features

* 💬 Real-time chat interface with message bubbles
* 🧠 AI-generated responses using Firebase AI API
* 🎨 Modern UI built with Jetpack Compose
* 👤 Distinct user and bot message styling
* ⚡ Dynamic message rendering with Compose state
* 📱 Clean and responsive Material design

---

## 🛠 Tech Stack

* **Language:** Kotlin
* **UI:** Jetpack Compose
* **Architecture:** MVVM
* **AI Integration:** Firebase AI API
* **State Management:** ViewModel + Compose State
* **IDE:** Android Studio


## ⚙️ Installation & Setup

1️⃣ Clone the repository

```bash
git clone https://github.com/Shreyjain-1/ChatBot.git
```

2️⃣ Open in **Android Studio**

3️⃣ Add your Firebase configuration (`google-services.json`)

4️⃣ Sync Gradle

5️⃣ Run on emulator or device

---

## 🔥 Firebase Setup

To run this project with AI responses:

1. Create a Firebase project
2. Enable Firebase AI API / Generative AI
3. Add `google-services.json` to app module
4. Sync and run
